## Usage Guidelines

### DO's ✅

- Use the provided logo files without modification
- Maintain proper spacing around the logo (minimum clear space)
- Use the white logo on dark backgrounds
- Use the black logo on light backgrounds
- Ensure logos are clearly visible and not obscured
- Use high-resolution formats (SVG preferred) for print materials

### DON'Ts ❌

- Do not modify, distort, or recreate the logo
- Do not change the logo colors
- Do not add effects (shadows, gradients, outlines)
- Do not rotate or skew the logo
- Do not place the logo on busy backgrounds that reduce visibility
- Do not use low-resolution versions for large formats

### Color Palette

CitreaScan's primary brand colors:

- **Orange**: `#F6741D`, `#F07823`, `#FFA217`
- **Black**: `#000000`
- **White**: `#FFFFFF`
