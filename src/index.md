# Welcome to CitreaScan Documentation

CitreaScan is the dedicated blockchain explorer for **Citrea Testnet** - Bitcoin's first ZK Rollup. Explore transactions, blocks, addresses, and smart contracts with a powerful, user-friendly interface.

## What is Citrea?

Citrea is a groundbreaking **Type 2 zkEVM** that brings Ethereum compatibility to Bitcoin through zero-knowledge technology. Every transaction on Citrea is secured by cryptographic proofs and settled on Bitcoin, combining Bitcoin's security with zkEVM's scalability.
